## 📊 Supply Chain \& Logistics Analysis using SQL



##### 📌 Project Overview

This project analyzes supply chain data using SQL to identify inefficiencies in lead time, shipping, cost, inventory, and product quality.







##### 🛠 Tools Used

* MySQL
* SQL







##### 📈 Key Analysis Performed

* Lead Time Analysis
* Shipping Performance Analysis
* Cost Analysis
* Inventory Risk Analysis
* Quality (Defect Rate) Analysis







##### 🔍 Key Insights



###### 📊 Lead Time Analysis

Haircare products have the highest average lead time (\~18.7 days), indicating inefficiencies in the supply chain.





###### 🚚 Shipping Performance

Shipping performance varies across carriers, suggesting the need for optimization in logistics.





###### 💰 Cost Analysis

Transportation modes differ in cost efficiency, indicating potential opportunities for cost reduction.





###### 📦 Inventory Risk

Some products have low stock levels despite high demand, indicating risk of stockouts.





###### 🧪 Quality Analysis

Haircare products have the highest defect rate, indicating possible quality control issues.





###### ⚠️ Overall Risk

Haircare is identified as the worst-performing category based on lead time, defect rate, and cost.







##### 🧠 SQL Concepts Used

\- GROUP BY

\- HAVING

\- Aggregate Functions (AVG, SUM)

\- Subqueries

\- ORDER BY

This project demonstrates how SQL can be used to extract meaningful business insights and support data-driven decision-making in supply chain management.
